Maze+XML - Maze Document

Description

The Maze+XML media type is an XML data format for sharing maze state information between clients and servers. It can be used to implement simple mazes, adventure games, and other related data.

NOTE:
The current version exposes only the basics of navigating a simple 2-D maze. Future additions to the media type will increase functionality w/o breaking support for the current feature set.

MIME Types

  • application/vnd.amundsen.maze+xml approved
