Link Relations - Registrations

Description

This page contains a list of link relation values registered by Mike Amundsen. Entries here are in various states of registration (draft, proposed, pending, approved, rejected, etc.).

NOTE
This is not a list of all the Link Relation Types used in the media types designed/registered by Mike Amundsen. Only the Link Relation Types that are targeted for IANA registration are listed here.

1.

Item And Collection Link Relations

 

View current status HERE

The process of creating an Internet Draft (I-D) for the 'item' and 'collection' link relation types was initiatedin August of 2011. Below is the progress timeline.

  1. 2011-08-04 : Posted 'item' and 'collection' to Microformats Existing Link Rel Values page.
  2. 2011-08-05 : Posted 'item' and 'collection' to ParamsR.Us tracker site for IANA approval.
  3. 2011-08-05 : Application for IANA approval re-routed. I was asked to submit an I-D (supplied a basic template, etc.)
  4. 2011-08-06 : Posted pre-initial draft of I-D to web. Followed up with posting to link-relations mailing list for feedback.
  5. 2010-08-10 : After basic review and advice from folks on the email list, I submitted the "00" draft to the IETF Data Tracker. Passed validation first time (thanks to help from Mykyta & JReschke). Now time to wait for formal feedback via the list...
